# 🎮 Tinta Forma

**Tinta Forma** is a simple 2D puzzle-platformer developed as part of a game development course.

This project focuses on the fundamentals of platformer mechanics, basic level design, and player interaction. It was created as a learning experience using GameMaker, exploring core concepts such as movement, collision, and simple puzzle elements.

---

## 🕹️ Gameplay

In *Tinta Forma*, you control a character navigating through small levels filled with obstacles and light puzzle elements. The game emphasizes experimentation with movement and interaction rather than complex systems.

---

## ✨ Features

* Basic platformer mechanics
* Simple puzzle elements
* Minimalist design
* Educational project structure

---

## 🎮 Controls

* **A / Left Arrow** → Move left
* **D / Right Arrow** → Move right
* **Space** → Jump
* **E** → Ink Power
* **Shift** → Run Power
